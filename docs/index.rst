Welcome to vib's documentation!
===============================
Vib is a package for analyzing nonlinear vibrations.

The project is on `GitHub`_.

User Guide
----------

The vib user guide explains how to install vib, how to build and train
neural networks using vib, and how to contribute to the library as a
developer.
